About Flask Blog

Welcome to Flask Blog - a modern, secure blogging platform built with Python and Flask!

Features:
  • Secure Authentication: Traditional email/password login with secure password hashing
  • Social Login: Easy login with GitHub and Google OAuth
  • Rich Content: Create and publish blog posts with formatted content
  • User Management: Personal dashboard to manage your posts
  • Responsive Design: Beautiful, mobile-friendly interface
Technology Stack:
  • Backend: Python Flask with SQLAlchemy
  • Authentication: Flask-Login + Flask-Dance (OAuth)
  • Frontend: Bootstrap 5 + Font Awesome
  • Database: SQLite (configurable)
